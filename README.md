## âš™ï¸ Available Commands

All commands are run from the root of the project, from a terminal:

| Command | Action |
|---------|--------|
| `npm install` | Installs dependencies |
| `npm run dev` | Starts local dev server at `localhost:4321` |
| `npm run build` | Build your production site to `./dist/` |
| `npm run preview` | Preview your build locally, before deploying |
| `npm run astro ...` | Run CLI commands like `astro add`, `astro check` |
| `npm run astro -- --help` | Get help using the Astro CLI |

## ğŸ¨ Customization

### Personal Information

Edit your personal information directly in the Astro components and pages.

### Adding Projects

Add your projects in the `src/content/projects/` directory:

```markdown
---
title: "Project Name"
description: "Brief description"
image: "/images/project.jpg"
tags: ["React", "TypeScript", "Tailwind"]
link: "https://project-url.com"
---

Your project details here...
```

### Styling

Customize colors and styles in:

- `src/styles/global.css` - Global styles
- Individual component files - Component-specific styles

### Theme Colors

Update your color scheme in the CSS variables:

```css
:root {
  --primary: #your-color;
  --secondary: #your-color;
  --accent: #your-color;
}
```
